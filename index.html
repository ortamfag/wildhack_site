WildhackSupportBot

WildhackSupportBot - это чат-бот, созданный для помощи волонтерам, спасающим дикую природы России. Проблема помощи в сохранении исчезающих видов животных и в улучшении состояния дикой природы остро стоит уже длительное время.

Благодаря нашему чат-боту вы сможете больше узнать о проблеме конкретной области дикой природы нашей страны, сможете своевременно получать данные о том, в каком заповеднике требуется помощь сейчас

Актуальность проекта

Проблема помощи в сохранении исчезающих видов животных и в улучшении состоянии дикой природы остро стоит уже несколько лет.

Для повышения эффективности мер, направленных на сохранение дикой природы целесообразно использовать современные IT-разработки, которые повысят охват освещения проблемы и улучшат удобство информирования волонтеров

Функционал

Пользователь может начать работу с чат-ботом после написания команды «/start»

Важные сообщения помечаются иконкой

После каждого начала диалога в сообщении появляется случайная иконка животного

Возможность задать вопрос. В этом случае пользователь пишет своими словами интересующую его область, а бот постарается на этот вопрос ответить, исходя из файлов JSON

Выбор интересующих кордонов. Пользователю предлагается список кордонов. После выбора одного из них, бот рассказывает всю информацию об этом месте, в том числе сведения для волонтеров.

Приятного пользования!
